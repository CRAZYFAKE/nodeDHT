##nodeDHT:
A DHT crawler, is not quite based on Kademlia, written by NodeJS, the code is very very simple.

##dependencies:
`npm install`

##start nodeDHT
`node nodeDHT.js`

##notice:
You can edit `MAX_NODE_QSIZE` value for speed.
